EVENT BOOKING SYSTEM 🎟️

The Event Booking System is a microservices-based platform that allows users to browse, book, and manage event tickets seamlessly.

🔹 Key Microservices in the System

  • 📌 User Service: Manages user authentication.
  • 📌 Event Service: Handles event listings.
  • 📌 Booking Service: Processes ticket bookings.
  • 📌 Notification Service: Sends email confirmations.
